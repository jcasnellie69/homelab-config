# Contributing

Thanks for your interest in contributing to this repository. A quick checklist:

- Fork the repository and create a branch for your change.
- Follow the repository coding style (`.editorconfig`).
- Run linters locally: `markdownlint` for docs and `shellcheck` for shell scripts.
- Open a pull request with a clear description and testing steps.

If your change adds or modifies scripts, ensure executables have correct shebangs and permissions.
