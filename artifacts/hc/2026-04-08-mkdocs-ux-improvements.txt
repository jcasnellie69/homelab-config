## Summary
Added two MkDocs Material theme features to mkdocs.yml to improve the
documentation site UX:

  1. content.code.copy  — renders a copy-to-clipboard button on every
                          code block, saving time when copying commands
                          or scripts.
  2. navigation.path   — displays breadcrumbs so readers can see their
                          current location within the documentation
                          hierarchy.
