# Homelab Artifacts

Welcome to the homelab documentation and log viewer.

- This site is served by Nginx from `/srv/artifacts`.
- Markdown files in this directory (and subdirectories) can be linked and rendered here.
- Raw logs (`.log`, `.txt`, `.csv`) can be linked for download.

> Pro Tip: Your snapshot script can drop `.md` summaries into
> `pve-guest-snapshots/` and you can link them from the sidebar.
